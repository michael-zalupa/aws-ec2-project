# AWS EC2 Web Project

## 📋 Description
A web application with a database deployed on AWS EC2.

## 🛠 Tech Stack
- AWS EC2 (Amazon Linux 2023)
- Apache HTTP Server
- PHP
- MySQL (MariaDB)
- AWS IAM
- Git / GitHub

## 🚀 What was done
- Launched an EC2 instance via AWS Console
- Configured Apache web server
- Set up MySQL database
- Built PHP backend to save and display visitors
- Configured Security Groups (HTTP + SSH)
- Automated server setup with EC2 User Data Script

## 📦 Structure
├── index.php      # Main website file
├── userdata.sh    # Automated server setup script
└── README.md      # Documentation
